Динамические массивы в языке Pascal предоставляют гибкость для работы с массивами, которые могут изменять свой размер во время выполнения программы. Основная идея заключается в том, что память для динамического массива выделяется и освобождается по мере необходимости с помощью встроенных функций и процедур.

Основные концепции создания динамических массивов:

1.          Объявление динамического массива: В отличие от статических массивов, размер которых известен на этапе компиляции, динамические массивы объявляются без указания их размера. В Pascal для этого используется специальный синтаксис.

Var

dynamicArray: array of Integer;

2.          Выделение памяти:

Для выделения памяти под динамический массив используется процедура SetLength. Она задает размер массива и автоматически выделяет необходимую память.

SetLength(dynamicArray, 10);  // Динамический массив из 10 элементов

3.          Освобождение памяти:

Для освобождения памяти, занятой динамическим массивом, достаточно присвоить массиву нулевой размер или же вызвать SetLength с нулевым значением.

SetLength(dynamicArray, 0);  // Освобождение памяти

Заполнение одномерного динамического массива

Заполнение с использованием цикла:

1.          Заполнение массива с помощью цикла — самый распространенный способ, особенно когда значения вычисляются или вводятся последовательно.

program FillOneDimArray;

var

  arr: array of Integer;

  i: Integer;

begin

  SetLength(arr, 10);  // Выделение памяти для массива из 10 элементов

  for i := 0 to High(arr) do

  begin

    arr[i] := i * 2;  // Пример заполнения: каждый элемент равен удвоенному индексу

  end;

 

  // Вывод значений массива

  for i := 0 to High(arr) do

  begin

    WriteLn('Элемент ', i, ' = ', arr[i]);

  end;

end.

2.    Заполнение с использованием пользовательского ввода:

Заполнение массива данными, введенными пользователем, может быть полезно в интерактивных приложениях.

program FillArrayWithUserInput;

var

  arr: array of Integer;

  i, n: Integer;

begin

  Write('Введите количество элементов: ');

  ReadLn(n);

  SetLength(arr, n);  // Выделение памяти для массива из n элементов

 

  for i := 0 to High(arr) do

  begin

    Write('Введите элемент ', i, ': ');

    ReadLn(arr[i]);  // Чтение значения элемента массива от пользователя

  end;

 

  // Вывод значений массива

  for i := 0 to High(arr) do

  begin

    WriteLn('Элемент ', i, ' = ', arr[i]);

  end;

end.

3.          Заполнение с использованием стандартных функций или процедур:

Иногда массив заполняется значениями, полученными из стандартных функций или процедур.

program FillArrayWithRandomValues;

var

  arr: array of Integer;

  i: Integer;

begin

  Randomize;  // Инициализация генератора случайных чисел

  SetLength(arr, 10);  // Выделение памяти для массива из 10 элементов

 

  for i := 0 to High(arr) do

  begin

    arr[i] := Random(100);  // Заполнение массива случайными значениями от 0 до 99

  end;

 

  // Вывод значений массива

  for i := 0 to High(arr) do

  begin

    WriteLn('Элемент ', i, ' = ', arr[i]);

  end;

end.

Заполнение двумерного динамического массива

1.          Заполнение с использованием вложенных циклов: Для заполнения двумерного массива используется пара вложенных циклов.

program FillTwoDimArray;

var

  arr: array of array of Integer;

  i, j: Integer;

begin

  SetLength(arr, 3, 3);  // Выделение памяти для массива размером 3x3

 

  for i := 0 to High(arr) do

  begin

    for j := 0 to High(arr[i]) do

    begin

      arr[i][j] := i * j;  // Пример заполнения: элемент равен произведению индексов

    end;

  end;

 

  // Вывод значений массива

  for i := 0 to High(arr) do

  begin

    for j := 0 to High(arr[i]) do

    begin

      Write(arr[i][j]:4);

    end;

    WriteLn;

  end;

end.

2.          Заполнение с использованием пользовательского ввода:

Пользовательский ввод может использоваться для заполнения двумерного массива.

program FillTwoDimArrayWithUserInput;

var

  arr: array of array of Integer;

  i, j, rows, cols: Integer;

begin

  Write('Введите количество строк: ');

  ReadLn(rows);

  Write('Введите количество столбцов: ');

  ReadLn(cols);

  SetLength(arr, rows, cols);  // Выделение памяти для массива размером rows x cols

 

  for i := 0 to High(arr) do

  begin

    for j := 0 to High(arr[i]) do

    begin

      Write('Введите элемент [', i, ', ', j, ']: ');

      ReadLn(arr[i][j]);  // Чтение значения элемента массива от пользователя

    end;

  end;

 

  // Вывод значений массива

  for i := 0 to High(arr) do

  begin

    for j := 0 to High(arr[i]) do

    begin

      Write(arr[i][j]:4);

    end;

    WriteLn;

  end;

end.

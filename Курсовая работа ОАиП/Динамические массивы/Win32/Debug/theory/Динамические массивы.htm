Динамические массивы в Pascal ABC.NET

Динамическим называется массив, размер которого, при необходимости, может меняться во время исполнения программы.

Динамические массивы дают возможность более гибкой работы с данными, так как позволяют не прогнозировать хранимые объёмы данных, а регулировать размер массива в соответствии с реально необходимыми объёмами.

 В отличие от динамических массивов существуют статические массивы и массивы переменной длины. Размер статического массива определяется на момент компиляции программы.

Размер массива переменной длины определяется во время выполнения программы. Отличием динамического массива от массива переменной длины является автоматическое изменение размеров, что не трудно реализуется в случаях его отсутствия, поэтому часто не различают массивы переменной длины с динамическими массивами.

Описание динамического массива

Тип динамического массива конструируется следующим образом:

 

array of тип элементов  (одномерный массив)
array [,] of тип элементов  (двумерный массив)
и т.д.

Переменная типа динамический массив представляет собой ссылку. Поэтому динамический массив нуждается в инициализации (выделении памяти под элементы).

 

Выделение памяти под динамический массив

Для выделения памяти под динамический массив используется два способа.

1.      Первый способ использует операцию new в стиле вызова конструктора класса:

var
  a: array of integer;
  b: array [,] of real;
begin
  a := new integer[5];
  b := new real[4,3];
end.

Данный способ хорош тем, что позволяет совместить описание массива и выделение под него памяти:

var
  a: array of integer := new integer[5];
  b: array [,] of real := new real[4,3];

Описание типа можно при этом опускать - тип автовыводится:

var
  a := new integer[5];
  b := new real[4,3];

2.      Второй способ выделения памяти под динамический массив использует стандартную процедуру SetLength:

 

SetLength(a,10);
SetLength(b,5,3);

Элементы массива при этом заполняются значениями по умолчанию.

Процедура SetLength обладает тем преимуществом, что при ее повторном вызове старое содержимое массива сохраняется.

 

 

 

 

 

 

Пример № 1. Динамические одномерные массивы ( ВВОД, ВЫВОД и определение размера одномерного динамического массива)  Версия для PascalABC.NET.

 

var A: array of integer;

    i, N: integer;

 

procedure printArray ( X: array of integer); var i: integer; begin

  writeln ( 'Массив:' );

  for i:=0 to High(X) do

    write ( X[i], ' ' );

  writeln;

  writeln ( 'Размер массива: ', Length(X) ); end;

 

begin

  write ( 'Введите размер массива: ' ); 

 readln ( N ); 

 SetLength ( A, N );

  for i:=0 to N-1 do

    A[i] := i;

 

  printArray ( A );

 

end.

 

 

ПРИМЕР №2  Работа с одномерным динамическим массивом.  Расширение динамических массивов 

C клавиатуры вводятся натуральные числа, ввод заканчивается числом 0. Нужно вывести на экран эти числа в порядке возрастания.  

 

program dynamicArraysExt;

var A: array of integer; i, j, x, N, c: integer;

 

procedure printArray ( X: array of integer; N: integer);

var i: integer;

 begin   for i:=0 to N-1 do    

   write ( X[i], ' ' );

  writeln;

end;

 

begin

   SetLength(A, 2);

   writeln ( 'Вводите числа (0 - закончить ввод): ' );

   N:= 0;   read(x);

  while x <> 0 do begin   

if N > High(A) then

   begin      

   SetLength(A, Length(A)+2);

      writeln('Размер массива увеличен до ', Length(A));    

      end;    

      A[N]:= x;   

      N:= N + 1;   

      read(x) 

      end;

  readln;

 

  writeln ( 'Массив до сортировки:' );

  printArray ( A, N );

  for i:=0 to N-2 do   

     for j:=N-2 downto i do     

        if A[j] > A[j+1] then begin      

        c:=A[j];

        A[j]:=A[j+1];

        A[j+1]:=c     

        end;

 

  writeln ( 'Массив после сортировки:' );

  printArray ( A, N );

 

end.
